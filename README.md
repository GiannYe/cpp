# rendimento_studenti.cpp
Il programma crea una matrice di n righe, scelte dall'utente, che rapressentano il numero di studenti. Crea voti casuali per 10 materie e calcola la media delle materie a ogni studente. Trova la media maggiore. In fine permette all'utente di trovare dei voti simili (voti che hanno la differenza di 0.5) al voto che l'utente inserisce in input

# AlgoritmiDiOrdinamento.cpp
Il programma conta il numero di assegnazioni e di confronti di 3 algoritmi di ordinamento ingenui: insection, selection e bubble sort

# AlgoritmiDiOrdinamento-GianniYe.pdf
Relazione che spiega ogni riga del programma AlgoritmiDiOrdinamento.cpp

# cinema.cpp
Il programma simula un concorso cinematografico in cui 100 giudici valutano 25 film. Ogni giudice sceglie casualmente 10 tra i 25
