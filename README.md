# rendimento_studenti.cpp
Il programma crea una matrice di n righe, scelte dall'utente, che rapressentano il numero di studenti. Crea voti casuali per 10 materie e calcola la media delle materie a ogni studente. Trova la media maggiore. In fine permette all'utente di trovare dei voti simili (voti che hanno la differenza di 0.5) al voto che l'utente inserisce in input
